# Web Client Project

## Description
Web application with authentication, playlists management, MP3 upload and YouTube integration.

## Technologies
- Node.js + Express
- Sessions authentication
- Multer (MP3 upload)
- YouTube Data API (secured on server)
- Vanilla JS / HTML / CSS

## Architecture
- server: backend (routes, services, middleware)
- client: frontend (pages, scripts, styles)

## Security
- API keys stored in .env
- Passwords hashed with bcrypt
- Session-based authentication

## Run
npm install  
npm start  
Open http://localhost:3000
